# 1 前言

RoboMaster 参赛队伍需自行研发和制作参赛机器人。参赛机器人需满足本文档描述的所有规范，检录工作人员会根据本文条例对机器人进行抽查。此外，参赛机器人需符合相关法律法规，不得存在安全隐患，若因违规导致安全事故，组委会将依法追究违规方的法律责任。若本规范存在争议之处，以裁判长或检录长的解释为准。 

为便于参赛队伍理解并遵守本手册条例，特此说明条例的约束力与可能后果： 

- 全文中凡在条目前标注 “S#”（例如：S10）的条例，均为强制性条例，强制性条例侧重于安全、尺寸/重量、电磁兼容性、场地规则等关键要求，必须严格遵守。违反强制性条例，组委会有权拒绝该机器人参赛、要求限期整改或直接判定失去比赛资格，具体处理视违规性质而定。 

- 未标注“S#”的条例为建议性条例，建议性条例用于优化性能、提高稳定性或减少故障风险，但不作为检录合格的硬性条件。未遵循建议性条例，虽不直接影响检录通过，但可能导致模块功能异常、比赛中性能受限、发生额外的维修/定损费用或产生安全隐患；因此由此产生的一切损失与后果由参赛队伍自行承担。 

除非在条例中另有说明，本手册中所有强制性条例均适用于 RMUC 与 RMUL 参赛机器人。若某项条例涉及的机器人类别、场地道具或比赛规则与 RMUL 无关，则 RMUL 参赛机器人可被豁免该项具体要求。

# 2.1.1 能源

S1 机器人使用的能源形式仅限于电源和气源。 

S2 对于受底盘功率限制的机器人，其底盘水平方向移动的动力源仅限于电源。

- 禁止使用液压或其他有可能产生污染的动力方式。

# 2.1.1.2 气源

S8 气瓶中存储的压缩气体气压不得大于 20Mpa，气瓶标称耐压值不得小于 30Mpa，且在该气瓶出口直接安装双表恒压阀，工作气压不得大于 0.8Mpa。 

S9 工作气体需满足不可燃烧、无毒、无污染的条件，如空气、氮气、二氧化碳。 

S10 任何使用或存储气体的机构（气瓶、气缸、空气弹簧等）需要具有合格证书或铭牌钢印，检录时合格证书、铭牌钢印需方便查看。 

S11 若气瓶尚在寿命使用期内，需按说明书或铭牌规定的期限返厂维护，并且提供维护证明。 

S12 气瓶需具备由原产地国家官方认可的认证机构颁发的认证证书。 

S13 气瓶需安全稳固地安装于机器人机体上，瓶体至少有 2 个距离大于 1/5 瓶身长度的固定点或必须有大于 1/5 瓶身长度的固定面。气瓶及输气管需施以保护，以防止因机器人侧翻、碰撞、旋转、运动部件故障等造成损坏。瓶口不得外露，防止被弹丸击打造成损坏。为确保安全，瓶口必须保持水平或朝上。 

S14 气瓶安装时需考虑机器人以任意角度翻滚，气瓶和输气管都不会接触到地面。 

S15 输气管和配件需适用于系统可能的最大工作压力。

# 2.2.1 充分再设计

充分再设计是指对机器人核心机构的改进，相关系统的牵连较为广泛。 

# 1. 底盘

● 轮组方案改变。例如：麦克纳姆轮、舵轮、全向轮、阿克曼转向轮、履带等方案的改变； 

● 轮组传动方案改变。例如：动力在簧下，动力在簧上等方案的改变； 

- 悬架方案改变。例如：双叉臂悬架、扭力梁悬架、双纵臂悬架、升降悬架等方案的改变； 

● 底盘动力改变。例如：直流、交流，有刷、无刷，减速、直驱等动力改变； 

● 底盘形式改变。例如：框架式、板层式、承载式等底盘形式的改变。 

# 2. 云台

- 云台传动堆叠改变。例如：Yaw、Pitch、Roll 轴包含关系、数量发生变化； 

● 供弹原理变更。例如：机器人供弹链路由简单直连供给变更为通过 Yaw 轴关节供给至发射机构； 

- 弹仓位置变化。例如：弹仓和底盘固连、Yaw 轴固连、Pitch 轴固连等方式的改变； 

- 云台电机传动方式变化。例如：直驱、带轮、连杆、齿轮等方式的改变（不包括带轮、齿轮、链轮之间的互换）。 

# 3. 执行机构

- 除末端执行器的其他机构，拓扑结构的变化，例如：转动副、滑动副等的组合顺序、数量、类型改变； 

- 执行机构的动力效率提升，例如：功能不变的情况下动力数量减少、有效功能增加的情况下动力数量不变、有效功能增加的情况下动力数量增加。 

# 4. 飞镖系统

● 飞镖：空气动力学原理上的重新设计。包括机翼构型、气动外形、动力方式、稳定方式等。 

- 飞镖发射架：在核心传动系统或发射方式方面上的重新设计，包括弹射方式、动力传递结构、推送机构布局等。 

![](images/a4b4a0811ec49b315ac3a9321bc57ecaed9b5d5e146307a3c5ee6698a6f53d28.jpg)


飞镖与飞镖发射架的充分再设计要求相互独立，均需满足。 

# 2.2.2 非充分再设计

非充分再设计是指与相关系统牵连较少的改进。 

- 悬架硬点参数变化幅度大于 $10\%$ ; 

● 有效动力轮组数量变化; 

● 轮距、轴距的变化大于 10%; 

● 底盘动力电机减速比改变; 

● 裁判系统机载端模块放置方位变化至少 3 个模块，且均导致机器人系统出现拓扑结构变化； 

- 增加与机器人原功能相关联的新功能模块。例如：独立救援装置、图传小转台、能量单元转向、视觉模块的增加； 

● 摩擦轮纵置和横置变化; 

- 云台电机传动方式变更。例如：带轮、齿轮、链轮之间的互换； 

● 动力源的变更。例如：气动、电动； 

- 末端执行器的类型、方案变更。例如：旋转夹取，变为平移夹取； 

● 核心电子器件布局（主控、电源、运算平台、感知模块）。例如：位置从云台切换到底盘； 

- 云台旋转范围从有限变为无限； 

- 飞镖局部气动构件尺寸、数量或安装角度的调整，例如尾翼面积变化、尾翼安装角度变化、翼型弯度调整等； 

● 飞镖材料更换，并在质量分布或表面特性方面进行调整； 

● 飞镖发射架供弹机构布局或结构形态的调整，如供弹轨道形状改变、推弹机构形状优化等； 

● 发射架部件间传动比或传动路径的修改。 

![](images/20cdfe73e562c8ee4ed56555247615201f19ee39e0cd28405bed78ccddeb651d.jpg)


飞镖与飞镖发射架的非充分再设计要求相互独立，均需满足。 

# 2.2.3 无效再设计

无效再设计是指对核心功能影响较小的改动。 

无效再设计包括但不限于以下改动： 

● 非主要承重结构、遮罩防护结构改动。例如：镂空发生变化、形状发生变化； 

● 标准件型号数量的变更; 

● 气缸和电推杆互换; 

- 飞镖挂载方式的重新设计； 

- 同特性材质的变换。例如：玻纤板和碳纤板互换； 

- 原理无变化的变更。例如：上供弹硬管改软管等硬度的改变。 

# 2.3 第三方成品模组的定义和适用范围

S91 除另有规定外，参赛队伍在制作机器人时，禁止使用第三方成品模组。 

- 第三方：除组委会、参赛队伍自身之外的其他主体。 

- 成品模组：由数个基础功能组件组成的特定功能组件，可用来组成具备完整功能的系统，例如：机械臂、底盘、超级电容控制板及配套代码、云台、飞镖、带悬挂轮组、拨弹机构、发射机构。 

空中机器人的飞行系统（含机架、动力系统、飞控、感知系统）、雷达运算平台端和自定义客户端所使用的运算平台可以使用成品模组，不受该条例限制。

S115 飞镖需定位飞镖检测模块，通过螺旋桨（最多允许使用 1 个）、舵面、喷气等方式控制飞行方向，最终撞击飞镖检测模块实现击打效果。飞镖推重比需小于 1。 

S116 飞镖在发射时飞镖触发装置需亮起，且在俯视视角下观测飞镖系统不得存在除裁判系统和飞镖触发装置外的外露光源。

# 3.6.1.1 装甲模块安装

S149 机器人进行装甲模块安装时，装甲模块和装甲支撑架需稳固连接。装甲模块的装甲支撑架底部连接面需与 XY 平面平行，使得装甲模块受力面所在平面的法向量所在直线与 Z 轴负方向所在直线的锐角夹角为 $75^{\circ}$ 。装甲模块不含侧灯的两条边与 XY 平面夹角小于 $5^{\circ}$ 。定义安装好的装甲模块受击打面的法向量（与 Z 轴负方向夹角为锐角）在 XY 平面上的投影为该装甲模块的方向向量。4 块装甲模块的方向向量需分别与机器人机体坐标系的正 X 轴、负 X 轴、正 Y 轴、负 Y 轴一一对应，方向向量和对应坐标轴之间的角度误差不能超过 $5^{\circ}$ 。 

S150 机器人本身的运动学方程也需建立在上述机体坐标系下。装甲模块的安装方式需与机器人本身的结构特性或者运动学特性共享同一个参考坐标系。X轴上安装的装甲模块几何中心点连线与Y轴上安装的装甲模块几何中心点连线要互相垂直。安装于Y轴的单个装甲模块在发生偏移后，其几何中心相对于机器人的几何中心的偏移量在X轴上的分量不得超过 $50\mathrm{mm}$ ，安装于X轴上的装甲模块同理。

# 3.6.1.3 机器人变形

S152 原则上，比赛开始后，任何一块装甲模块不能主动地相对于机器人整体的质量中心发生移动。如果参赛机器人因为机器人结构设计需求导致机器人具有可变形特性，其装甲模块需满足：任何时候，任何一块装甲模块不得相对于机器人底盘发生连续、往复的快速移动。快速移动定义为移动速度超过 0.5m/s。

S184 电容电压需要由满电降低至 1V 以下，如果电压出现异常快速降低等现象将无法通过检录。 

S185 超级电容管理模块的检录接口需引出至少 100mm 长的 XT30 母头接口的线材。 

S186 英雄机器人、步兵机器人、哨兵机器人均需安装超级电容管理模块。若机器人没有超级电容模组，则只需采用 4-pin 线将超级电容管理模块接入电源管理模块。 

S187 超级电容管理模块需安装在机器人容易操作的位置，以便检录时操作超级电容管理模块。 

S188 超级电容模组仅允许使用一个供电接口，且需按照“图 3-50 超级电容管理模块接线示意图”进行连接，不得连接机器人上的其他电路。
